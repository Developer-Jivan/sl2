# 7. Sigmoid Function

```python id="e0k4x5"
def sigmoid(x):
    return 1 / (1 + np.exp(-x))
```

---

## Purpose

Converts any number into:

```text id="zbq6y0"
0 to 1
```

Example:

```text id="n6g2iy"
10   -> 0.999
0    -> 0.5
-10  -> 0.000
```

---

# 8. Sigmoid Derivative Function

```python id="h8v2qz"
def sigmoid_derivative(x):
    return x * (1 - x)
```

---

## Purpose

Used in **Backpropagation**

It tells:

```text id="mrv7sn"
How much should weights change?
```

---

## Why formula uses x*(1-x)?

For sigmoid output:

```text id="mhh4p5"
Derivative = sigmoid(x) × (1 - sigmoid(x))
```

Since `O` and `H` are already sigmoid outputs, we use:

```text id="rm0a1h"
x * (1 - x)
```

---

# 9. Training Loop

```python id="3zjwrh"
for i in range(10000):
```

Train 10,000 times.

Each loop = one epoch.

# 10. Forward Propagation

Input moves forward.

---

## Hidden Layer

```python id="k54w8v"
H = sigmoid(np.dot(X, W1) + b1)
```

Formula:

```text id="a1ubdc"
H = sigmoid(X·W1 + b1)
```

Meaning:

* Multiply inputs by weights
* Add bias
* Apply sigmoid

---

## Output Layer

```python id="gsc4yx"
O = sigmoid(np.dot(H, W2) + b2)
```

Formula:

```text id="ndsqwy"
O = sigmoid(H·W2 + b2)
```

This gives final prediction.

---

# 11. Error Calculation

```python id="9j4d0x"
error = Y - O
```

Compare:

```text id="bxj2j4"

```

Example:

```text id="2l6z8f"

```

---

# 12. Back Propagation

Now network learns from mistake.

---

## Output Gradient

```python id="3g6a5m"
dO = error * sigmoid_derivative(O)
```

Means:

* take error
* multiply by slope of sigmoid

This decides output correction.

---

## Hidden Gradient

```python id="9m0kcv"
dH = np.dot(dO, W2.T) * sigmoid_derivative(H)
```

Output error is sent backward to hidden layer.

---

# 13. Update Weights

## Output weights

```python id="hq7h4s"
W2 = W2 + np.dot(H.T, dO) * 0.1
```

## Hidden weights

```python id="6plg2n"
W1 = W1 + np.dot(X.T, dH) * 0.1
```

---

# Why `0.1`?

Learning rate.

Controls step size.

```text id="s67l8h"
small = slow learning
large = unstable
```

---

# 14. Update Bias

```python id="1v1r3i"
b2 = b2 + dO.sum(axis=0) * 0.1
b1 = b1 + dH.sum(axis=0) * 0.1
```

Bias also learns.

# The Story of Backpropagation

```text id="fl2p4g"
1. Find mistake
2. Fix output layer
3. Send mistake backward
4. Fix hidden layer
5. Update weights
6. Repeat
```

That’s all these lines mean.
